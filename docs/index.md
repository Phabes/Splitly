# Project Documentation

Welcome to the project documentation. This repository tracks the development, architecture, and roadmap of the application.

## Overview

This application is a mobile app designed to manage groups, track shared expenses (or trips), and manage friends. It features a robust client-server architecture with real-time state management and custom backend data caching.

---

## Tech Stack

### Frontend

- **Framework:** React Native (Expo)
- **Navigation:** React Navigation (Nested Tab/Stack architecture)
- **UI Components:** Custom components
- **Icons:** FontAwesome (`@fortawesome/react-native-fontawesome`)

### Backend

- **Environment:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT (JSON Web Tokens) with Bearer strategy
